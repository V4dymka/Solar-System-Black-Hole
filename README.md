# Solar System + Black Hole

This project is a 3D visualization of the Solar System built with [Three.js](https://threejs.org/).  
It includes the Sun, planets with their orbits, and a black hole for an extra cosmic effect.

## Features
- Interactive 3D scene with orbit controls
- Realistic planet textures and orbital paths
- Black hole with particle dust effect
- Minimalist design, optimized for performance

## Demo
Open `index.html` in your browser or view it via GitHub Pages.
